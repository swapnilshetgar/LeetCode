# LeetCode Journey 🚀

Welcome to my LeetCode repository!

This repository documents my journey of learning Data Structures and Algorithms (DSA) through LeetCode. My goal is to build strong problem-solving skills and prepare for technical interviews.

## Learning Roadmap

### Phase 1: Brute Force

In this phase, I focus on understanding the problem and writing a working solution without worrying about optimization.

Topics:

* Arrays
* Strings
* Basic Math
* Loops
* Recursion

### Phase 2: DSA & Optimized Solutions

After understanding the brute-force approach, I solve the same or similar problems using optimized algorithms and data structures.

Topics:

* Arrays
* Hash Maps
* Two Pointers
* Sliding Window
* Stack
* Queue
* Linked List
* Binary Search
* Trees
* Heap (Priority Queue)
* Graphs
* Dynamic Programming
* Greedy Algorithms
* Backtracking
